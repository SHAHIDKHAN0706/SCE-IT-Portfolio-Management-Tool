# Data Model

Each initiative record uses canonical fields:

- `id`, `name`, `capability`
- `portfolioName`, `valueStream`
- `funded`, `fundingStatus`, `fundingSource`
- `driver`, `recommendation`, `bcr`
- `totalCapitalCost`, `year2026`, `year2027`, `year2028`
- `goLive`, `ouSponsor`, `outcomes`, `classification`, `dependsOn`

All analytics and PPT exports are computed from the currently filtered in-memory dataset.
